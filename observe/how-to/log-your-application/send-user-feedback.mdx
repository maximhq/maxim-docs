## Add feedback to traces

<CodeGroup>
    ```typescript JS/TS
    trace.feedback({
        score: 5,
        feedback: "Great job!",
        metadata: {
            "flow": "support",
            "properties" :{
                "name": "John Doe"
            }
        }
    });
    ```

    ```python Python
    from maxim.logger.components import Feedback

    trace.feedback(feedback=Feedback(score=5, comment="Great job!", metadata: {"flow": "support","properties" :{"name": "John Doe"}}))
    ```

    ```go Go
    trace.SetFeedback(&logging.Feedback{
        Score:   5,
        Comment: StrPtr("Great job!"),        
    })
    ```

    ```java Java
    Feedback feedback = trace.setFeedback(new Feedback(
        5,
        "Great job!"
    ));
    ```
</CodeGroup>
